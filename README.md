## О проекте
Удобный веб-интерфейс для просмотра и бронирования автомобилей в аренду. Проект представляет собой интерактивный каталог с возможностью выбора машины и перехода к оформлению заказа.

## Возможности сервиса
*   **Просмотр каталога** — все доступные автомобили представлены в виде карточек с фотографиями
*   **Выбор авто** — можно изучить внешний вид и характеристики каждой машины
*   **Бронирование** — интерактивное взаимодействие: клик по карточке открывает возможность забронировать выбранный автомобиль

## Интерфейс
Минималистичный и понятный дизайн: главный экран сразу показывает доступные автомобили, управление интуитивно — просто нажми на понравившуюся машину.
